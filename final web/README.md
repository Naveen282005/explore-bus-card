# SHUN 3D Bus Explorer — Blank Contact Edition

Open `index.html` and find `window.SHUN_CONTACT`.

Fill only:
- phone
- whatsapp (country code + number, digits only)
- instagram (full profile link)
- email
- location

Example format (do not copy these placeholders as real details):
phone: "+91 XXXXX XXXXX"
whatsapp: "91XXXXXXXXXX"
instagram: "https://www.instagram.com/YOUR_USERNAME/"
email: "YOUR_EMAIL"
location: "YOUR_LOCATION"

Once filled:
- Book This Bus buttons open WhatsApp with a pre-filled 21-Seater Coach Bus enquiry.
- Phone is tap-to-call.
- WhatsApp is clickable.
- Instagram opens the profile.
- Email opens a booking enquiry email.
- Contact details display automatically in the Contact section.

If WhatsApp is left blank, booking buttons scroll to Contact instead of opening a broken link.
